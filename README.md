# Epithet-ssh Tap

## How do I install these formulae?

`brew install epithet-ssh/tap/<formula>`

Or `brew tap epithet-ssh/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "epithet-ssh/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
